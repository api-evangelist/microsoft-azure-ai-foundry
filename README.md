# Azure AI Foundry (microsoft-azure-ai-foundry)
Azure AI Foundry REST API provides a unified platform for building generative AI applications. It supports model catalog access, prompt engineering, fine-tuning, RAG patterns, and responsible AI evaluation through a comprehensive set of management and data plane APIs.
